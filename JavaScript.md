##javascript

[noder-react-native](https://github.com/soliury/noder-react-native)


基于React-Native 和 Redux开发的手机APP

[vue](https://github.com/yyx990803/vue)

前端视图层库。本质上 Vue.js 做的是两件事情，数据绑定和视图层组件化。

[jquery-textcomplete](https://github.com/yuku-t/jquery-textcomplete)

一个可以对多种不同文本内容实现自动补全的 JS 插件, 包括 @ 用户名, 文本关键词和 Emoji 关键词等内容的自动补全, 配合上今天上午发布的 emojify.js, 可以使其成为 Web 端 Emoji 输入和显示的一整套快速解决方案

[simditor](https://github.com/mycolorway/simditor)

simditor 是 [@Tower-彩程](http://weibo.com/n/Tower-%E5%BD%A9%E7%A8%8B?from=feed&loc=at) 技术团队开发的一款富文本编辑器, 支持插件扩展, 可用其现有的插件完成内嵌 emoji, markdown 自动转换, @ 人自动补全等强大的功能

[emojify.js](https://github.com/Ranks/emojify.js)

一款能够将 emoji 关键词转换为 emoji 图片的 JavaScript 插件, 可以快速的为你的网站提供 emoji 表情支持
![](http://hassankhan.me/emojify.js/images/emoji/open_mouth.png)[demo地址](http://hassankhan.me/emojify.js/)

[clipboard.js](https://github.com/zenorocha/clipboard.js/)

一个不依赖于 Flash 的复制或剪切文本插件，使用最新的使用 execCommand() API，最重要的是仅仅 2KB 大小，除了 Safari 现代浏览器均已支持。
![](https://zenorocha.github.io/clipboard.js/)

[nude.js](http://)

一个 JS 裸体扫描器, 能够识别到图片中的人类是否穿着衣着
[官网](http://www.patrick-wied.at/static/nudejs/)

[Gatsby](https://github.com/gatsbyjs/gatsby)

可将纯文本转换到动态博客或者网站上, 是一个 React.js 静态网站生成器

[ZhiHuDaily-React-Native](https://github.com/race604/ZhiHuDaily-React-Native)

一个使用 React Native 实现的知乎日报 Android 兼 iOS 客户端
